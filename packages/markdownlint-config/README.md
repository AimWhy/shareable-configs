# Markdownlint Config

My personal shareable markdownlint configuration.

## Install

```bash
yarn add -D markdownlint @waldronmatt/markdownlint-config
```

## Usage

**`.markdownlint.json`**

```bash
{
  "extends": "@waldronmatt/markdownlint-config"
}
```

## Under The Hood

### `index.json`
